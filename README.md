# Task Management API (Laravel)

## Overview
This is a RESTful Task Management API built using Larave.
It allows performing CRUD operations on tasks and returns data in JSON format.
## Features
- Authentication using Laravel Sanctum
- Create tasks
- get all tasks
- Get single task
- update tasks
- Delete tasks
- Search tasks by title or description
- Filter tasks by status (copmleted/pending)
- Pagination support
- Input validation
- RESTful routing using apiResource

## Technologies Used
- PHP
- Laravel
- Laravel sanctum
- MySQL
## Usage Stepsa:
1. Register a new user
2. Login to receive an authentication token
3. Inclide the token in your request headers:
 Authorization : Bearer YOUR_TOKEN
## Testing the API 
You can test the API using:
. Postman
. Thender Client 
## Propose
This project was built to practice bulding secure and scalable RESTful APIs using Laravel and sanctum.
